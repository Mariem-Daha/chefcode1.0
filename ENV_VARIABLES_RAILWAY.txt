==============================================
ChefCode - Railway Environment Variables
==============================================

Copy these into Railway Dashboard → Variables tab:

==============================================
REQUIRED VARIABLES (App won't work without)
==============================================

OPENAI_API_KEY=sk-proj-your_openai_key_here
API_KEY=chefcode-secret-key-2024
ENVIRONMENT=production
ALLOWED_ORIGINS=*

==============================================
DATABASE (Auto-provided by Railway)
==============================================

DATABASE_URL=(automatically set when you add PostgreSQL)

==============================================
OPTIONAL - OCR Features (Invoice Scanning)
==============================================

PROJECT_ID=my-capstone-project-475113
LOCATION=us
PROCESSOR_ID=your_processor_id_here
GEMINI_API_KEY=your_gemini_api_key_here
GOOGLE_APPLICATION_CREDENTIALS={"type":"service_account","project_id":"..."}

Note: If you don't need OCR, you can skip these.

==============================================
HOW TO ADD VARIABLES IN RAILWAY
==============================================

1. Open your project in Railway dashboard
2. Click on your service
3. Go to "Variables" tab
4. Click "New Variable"
5. Add each variable (name and value)
6. Railway will auto-redeploy

==============================================
IMPORTANT NOTES
==============================================

✓ Don't use quotes around values in Railway
✓ DATABASE_URL is auto-set when you add PostgreSQL
✓ OPENAI_API_KEY must be valid for AI features to work
✓ API_KEY can be any secure string you choose
